# Lab1 – Windows Forms & WPF

## Опис роботи

Лабораторна робота №1 з дисципліни "Розробка інтерфейсів користувача".

Розроблено два застосунки для обчислення:

- Загальної суми надходжень за депозитом
- Загальної суми виплат за кредитом


### Windows Forms версія:
- Введення суми, ставки та терміну
- Розрахунок депозиту
- Розрахунок кредиту
- Валідація введених даних
- Збереження результатів у .txt файл

### WPF версія:
- Реалізовано аналогічний функціонал
- Використано XAML для створення інтерфейсу
- Обробка подій Click
- Збереження результатів у текстовий файл
